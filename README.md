# Installation

1. clone the repository:
`$ git clone https://github.com/MiKoronjoo/rainbow.git`
2. add rainbow to /usr/bin
`$ cp ./rainbow/rainbow /usr/bin/rainbow`
3. change the permissions:
`$ sudo chmod 777 /usr/bin/rainbow`
4. enjoy :)

# Usage

`$ {command} | rainbow`
